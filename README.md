# Personal Finance Assistant (Python)

A backend project built with FastAPI + SQLite for tracking, analyzing, and forecasting personal finance transactions.

## Features (planned)
- Import transactions from CSV
- Store and query transactions
- Categorization rules
- Budget tracking
- Analytics endpoints
- Forecasting
